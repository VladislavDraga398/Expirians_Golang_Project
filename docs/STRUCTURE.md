# Documentation Structure

Единая точка входа в документацию: **[TECHDOCS.md](TECHDOCS.md)**.

## Layout

```text
docs/
├── TECHDOCS.md                  # Единый хаб техдоки
├── README.md                    # Краткая навигация
├── INDEX.md                     # Полный список файлов
├── STRUCTURE.md                 # Правила структуры
├── quick-start.md
├── roadmap.md
├── guides/
├── architecture/
├── operations/
├── decisions/
└── templates/
```

## Naming Rules

- Файлы: `lowercase-with-dashes.md`
- Папки: `lowercase`
- Новые документы сначала добавляются в `docs/TECHDOCS.md`
